# EDA_Sales_Analysis
# Superstore Sales EDA

## Project Overview
This project explores retail sales data to uncover trends in sales performance,
customer segments, product categories, and regional distribution.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Sales are right-skewed with few high-value transactions
- Technology is the top-performing category
- West and East regions generate maximum revenue
- Strong seasonality observed in Q4

## Dataset
Superstore sales dataset
